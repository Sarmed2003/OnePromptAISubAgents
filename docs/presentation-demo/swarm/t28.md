# Swarm lane 28

## Manager Agent: Tour Group Rotation Scheduling

Swarm lane 28 implements a manager agent responsible for scheduling tour group rotations across museum exhibitions. This agent integrates exhibition narrative data from lane 17 and access control tags from lane 25 to generate optimized tour_schedule_spec outputs that define time slots, visitor capacity constraints, and specimen highlights for each rotation.

## Architecture Overview

The Manager agent in lane 28 orchestrates the scheduling workflow by:

1. **Input Processing**: Receives exhibition narrative specifications from lane 17 and access tag configurations from lane 25
2. **Schedule Generation**: Creates tour_schedule_spec with time slots, capacity limits, and specimen highlights
3. **Output Distribution**: Produces structured schedules consumed by lane 29 (coordinator) for multi-lab experiment scheduling and museum operations

## Schedule Specification Format

The tour_schedule_spec includes:

- **Time Slots**: Scheduled tour departure times with duration and group size
- **Capacity Constraints**: Maximum visitor counts per rotation based on access tags and facility limits
- **Specimen Highlights**: Featured specimens and exhibition points aligned with narrative themes
- **Access Requirements**: Tag-based restrictions and special access considerations

## Integration Points

**Upstream Dependencies**:
- Lane 17: Exhibition narrative data providing thematic content and specimen information
- Lane 25: Access tag definitions for capacity and security constraints

**Downstream Consumers**:
- Lane 29: Coordinator agent uses tour schedules for multi-lab experiment coordination and museum operations planning

## Manager Agent Responsibilities

The manager agent handles schedule optimization by balancing visitor flow, exhibition narrative coherence, and access control requirements. It ensures that each tour rotation aligns with specimen availability, facility capacity, and operational constraints while maintaining educational quality through narrative-driven specimen highlights.
