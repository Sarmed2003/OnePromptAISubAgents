# Swarm lane 01

## Museum Curator Agent: T-Rex Anatomy Discovery

Lane 01 initiates the swarm by establishing a museum curator agent that discovers and catalogs comprehensive T-Rex anatomy records from paleontological archives. This agent systematically identifies and structures fossil data including species identifiers, bone nomenclature, and precise anatomical measurements to create a canonical fossil dataset.

## Canonical Fossil Data Schema

The museum curator agent outputs structured records following this schema:

```json
{
  "species_id": "string (e.g., 'T_REX_001')",
  "species_name": "string (e.g., 'Tyrannosaurus rex')",
  "records": [
    {
      "bone_id": "string (e.g., 'FEMUR_L')",
      "bone_name": "string (e.g., 'Left Femur')",
      "measurements": {
        "length_cm": "number",
        "width_cm": "number",
        "depth_cm": "number",
        "weight_kg": "number"
      },
      "condition": "string (e.g., 'complete', 'partial', 'fragment')",
      "discovery_date": "string (ISO 8601 format)",
      "location": "string"
    }
  ]
}
```

## Responsibilities

The curator agent in lane 01:

1. **Discovery**: Scans paleontological archives and museum collections for T-Rex skeletal specimens
2. **Cataloging**: Records bone identifiers, anatomical names, and precise measurements
3. **Validation**: Ensures all records contain required fields; missing or incomplete records trigger halt signals to downstream lanes
4. **Output**: Produces a validated fossil dataset that flows to lanes 02–05 for cross-reference, validation, and anatomical modeling

## Error Handling

If critical records are missing or validation fails, lane 01 signals a halt condition that prevents lane 02 and dependent lanes from proceeding. This ensures data integrity across the entire swarm workflow.
