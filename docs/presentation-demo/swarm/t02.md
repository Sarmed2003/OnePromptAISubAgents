# Swarm lane 02

## Paleontologist Agent: Skeletal Data Cross-Reference System

Lane 02 consumes fossil records from lane 01 and performs systematic cross-referencing of skeletal data against established anatomical literature databases. The paleontologist agent validates morphological features, assigns confidence scores based on literature alignment, and enriches the dataset with citations to peer-reviewed sources and anatomical references. This enriched skeletal_data output includes a cross_references field containing literature citations, confidence metrics, and validation status. Should validation fail or confidence scores fall below acceptable thresholds, the agent raises an alert for lane 03 (research assistant) to flag potentially problematic specimens for manual review.

## Input Contract

- **Source**: Fossil records from lane 01
- **Format**: Fossil dataset with anatomical measurements and morphological descriptions
- **Requirements**: Complete skeletal data fields for cross-referencing

## Output Contract

- **Primary Output**: Enriched skeletal_data with cross_references field
- **Cross-References Structure**: Literature citations, confidence scores, validation status
- **Alert Mechanism**: Validation failures trigger lane 03 alerts for research assistant review

## Processing Pipeline

1. **Data Ingestion**: Receive fossil records from lane 01
2. **Literature Matching**: Cross-reference skeletal features against anatomical literature database
3. **Confidence Scoring**: Calculate confidence metrics based on literature alignment strength
4. **Citation Generation**: Attach relevant peer-reviewed sources and anatomical references
5. **Validation**: Verify data integrity and threshold compliance
6. **Alert Generation**: Flag anomalies and low-confidence records for lane 03 review
7. **Output Enrichment**: Return validated dataset with complete cross_references metadata

## Error Handling

Validation failures, missing literature matches, and confidence score anomalies trigger structured alerts that cascade to lane 03 for human expert intervention and decision-making.
