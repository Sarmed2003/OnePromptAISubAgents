# Swarm lane 18: Educator Agent for Interactive Quiz Content Generation

## Overview

Lane 18 is the Educator agent responsible for generating interactive quiz content from exhibition narrative data (lane 17) and anatomical data (lanes 04–05). This agent transforms curatorial and scientific information into engaging, pedagogically sound quiz modules that drive learner engagement through the web UI (lane 07).

## Contract

**Input**: exhibition_narrative (from lane 17) + rig_model (from lanes 04–05)

**Output**: quiz_module containing quiz_content_spec with questions, answers, and learning objectives

## Responsibilities

The Educator agent performs the following core functions:

1. **Content Analysis**: Parses exhibition narrative to identify key concepts, themes, and learning opportunities that align with the curatorial vision.

2. **Question Generation**: Creates diverse question types (multiple choice, true/false, short answer, image-based) that test comprehension at multiple cognitive levels (recall, understanding, application, analysis).

3. **Answer Validation**: Develops comprehensive answer keys with explanations that reference both the narrative context and anatomical data for scientific accuracy.

4. **Learning Objectives Definition**: Articulates clear, measurable learning outcomes for each quiz section aligned with educational standards and exhibition goals.

5. **Difficulty Calibration**: Adjusts question complexity based on learner profile data and adaptive feedback mechanisms to optimize engagement.

6. **Anatomical Accuracy Integration**: Ensures all quiz content accurately reflects the anatomical models and rigged data provided by lanes 04–05, maintaining scientific integrity.

## Output Structure

The quiz_module produced by lane 18 includes:

- **Quiz sections** mapped to exhibition narrative segments
- **Question objects** with metadata (type, difficulty, learning_objective_id)
- **Answer objects** with correct answers, distractors, and explanatory feedback
- **Learning objectives** with Bloom's taxonomy levels
- **Scoring rubrics** for different question types
- **Adaptive branching logic** for personalized learning paths

## Integration Points

- **Upstream**: Consumes structured exhibition_narrative from lane 17 and anatomical models from lanes 04–05
- **Downstream**: Produces quiz_content_spec consumed by lane 07 (web UI) for rendering interactive quizzes
- **Feedback Loop**: Receives learner interaction data to continuously refine question difficulty and effectiveness

## Implementation Notes

The Educator agent operates as part of the swarm architecture, coordinating with:

- **Lane 07 (Web UI)**: Receives quiz specifications and renders interactive experiences
- **Lane 17 (Curator)**: Accesses narrative context and thematic priorities
- **Lanes 04–05 (Anatomist/Rigger)**: Integrates anatomical accuracy and model specifications

This design ensures that quiz content is both pedagogically sound and scientifically rigorous, creating meaningful learning experiences that reinforce exhibition narratives through interactive engagement.
