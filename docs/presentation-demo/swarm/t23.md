# Swarm lane 23: Photographer Agent

## Overview

Lane 23 is the Photographer agent responsible for optimizing specimen image metadata to ensure high-quality, discoverable, and accessible visual assets across the specimen documentation system. This lane processes rendered assets from lane 06 and comparison charts from lane 19, extracting and enhancing metadata including resolution, color space, EXIF data, and alt text to produce a comprehensive `image_metadata_spec` with optimization parameters.

## Inputs

- **Lane 06 (Rendered Assets)**: High-quality specimen images and visual renderings
- **Lane 19 (Comparison Charts)**: Comparative analysis visualizations and reference materials

## Processing

The Photographer agent performs the following operations:

1. **Resolution Analysis**: Extracts and validates image resolution, recommending optimal dimensions for different use cases (web, print, archival)
2. **Color Space Detection**: Identifies and documents the color space (sRGB, Adobe RGB, CMYK) and color profile information
3. **EXIF Data Extraction**: Parses embedded EXIF metadata including camera settings, capture date, lens information, and geolocation
4. **Alt Text Generation**: Creates descriptive, SEO-optimized alternative text for accessibility and search engine indexing
5. **Metadata Optimization**: Consolidates findings into a structured `image_metadata_spec` containing optimization parameters and recommendations

## Output

- **image_metadata_spec**: Structured metadata specification including:
  - Resolution specifications for different deployment contexts
  - Color space and profile information
  - Complete EXIF data catalog
  - Accessibility alt text
  - Optimization recommendations

## Storage

The generated `image_metadata_spec` is persisted to DynamoDB (lane 10) for reliable, queryable storage and downstream access.

## Downstream Integration

Lane 23 output feeds directly into:

- **Lane 24 (Librarian)**: Uses metadata for cataloging, organization, and publication workflow
- **Lane 07 (Web Deployment)**: Consumes metadata for web-optimized image serving, accessibility compliance, and SEO implementation

## Quality Standards

- All metadata must be accurate and complete
- Alt text must be descriptive, concise, and SEO-appropriate
- EXIF data preservation must maintain provenance and authenticity
- Optimization recommendations must balance quality, file size, and performance
