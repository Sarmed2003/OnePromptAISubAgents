# Swarm lane 19: Illustrator Agent

## Overview

Lane 19 is the Illustrator agent responsible for designing anatomical comparison charts that visualize morphological differences between prehistoric creatures (T-Rex, Velociraptor, and Pteranodon). This lane synthesizes symmetry metrics from lane 14 and anatomical data from lanes 04, 12, and 13 to produce detailed visual specifications that guide downstream processing.

## Responsibilities

### Input Sources
- **Lane 14**: Symmetry metrics and bilateral analysis for anatomical structures
- **Lane 04**: Core anatomical measurements and skeletal data
- **Lane 12**: Muscular and soft tissue anatomical specifications
- **Lane 13**: Behavioral and biomechanical constraints affecting visual representation

### Primary Deliverable

The lane produces a **comparison_chart_spec** artifact containing:
- Visual layout specifications for multi-specimen comparison
- Data bindings linking anatomical measurements to visual elements
- Symmetry annotations highlighting bilateral relationships
- Scale and proportion guidelines for accurate cross-species comparison
- Color coding and visual hierarchy for different anatomical systems

## Output Specifications

The comparison_chart_spec includes:
1. **Layout Grid**: Positioning of three specimens (T-Rex, Velociraptor, Pteranodon) for side-by-side comparison
2. **Anatomical Overlays**: Skeletal structures, muscle groups, and key morphological features
3. **Measurement Annotations**: Quantified differences in size, proportion, and symmetry metrics
4. **Visual Hierarchy**: Primary features emphasized, secondary structures contextualized
5. **Data Bindings**: Direct references to source measurements enabling dynamic updates

## Downstream Integration

- **Lane 23 (Photographer)**: Receives comparison_chart_spec for image optimization, rendering, and visual enhancement
- **Lane 24 (Librarian)**: Receives finalized assets for cataloging, publication preparation, and distribution

## Design Principles

- **Accuracy**: All visual representations grounded in peer-reviewed anatomical data
- **Clarity**: Symmetry relationships and key differences immediately apparent to viewers
- **Scalability**: Layout adapts to different display contexts and resolutions
- **Interoperability**: Chart specifications compatible with standard visualization tools and formats
