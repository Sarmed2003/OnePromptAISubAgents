# Swarm lane 20: Linguist Agent

## Overview

Lane 20 is the Linguist agent responsible for translating species names and anatomical descriptions for multilingual museum access. This agent enables global reach by providing translation bundles consumed by the web UI (lane 07) and educator resources (lane 18).

## Inputs

The Linguist agent receives data from three upstream lanes:

- **Lane 01 (Fossil Records)**: Raw fossil specimen data including original species nomenclature and taxonomic information
- **Lanes 04–05 (Anatomical Terms)**: Detailed anatomical descriptions and specialized terminology for fossil features
- **Lane 17 (Exhibition Narrative)**: Curatorial narrative text and contextual descriptions for museum exhibitions

## Processing

The Linguist agent performs the following operations:

1. **Species Name Translation**: Translates scientific and common names into multiple target languages while preserving taxonomic accuracy and maintaining standardized nomenclature conventions
2. **Anatomical Description Translation**: Converts specialized anatomical terminology into accessible descriptions across multiple languages, ensuring technical precision
3. **Cultural Adaptation**: Adapts exhibition narratives for cultural context and regional educational standards
4. **Quality Assurance**: Validates translations for consistency, accuracy, and domain-specific terminology alignment

## Outputs

The Linguist agent produces a `translation_bundle` containing:

- Translated species names (scientific and common names in multiple languages)
- Translated and localized anatomical descriptions
- Translated exhibition narratives
- Language metadata and encoding information
- Translation quality indicators and source-target language pairs

## Downstream Consumers

- **Lane 07 (Web UI)**: Consumes translation bundles to render multilingual museum interfaces and specimen information
- **Lane 18 (Educator Resources)**: Uses translated descriptions and narratives for educational materials and teacher resources

## Technical Considerations

The Linguist agent must handle:

- UTF-8 encoding for all text processing
- Multiple target languages with varying character sets and writing systems
- Preservation of scientific nomenclature accuracy across translations
- Consistency in terminology across specimen records and anatomical descriptions
- Fallback mechanisms for unsupported language pairs
