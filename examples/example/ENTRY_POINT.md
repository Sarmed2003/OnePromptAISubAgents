# Entry Point

## What This Project Is
A todo list REST API built with Node.js and Express. The goal is a clean, minimal backend that demonstrates CRUD operations with proper validation and error handling.

## How To Run
```bash
npm install
npm start        # starts server on port 3000
npm test         # runs jest test suite
```
