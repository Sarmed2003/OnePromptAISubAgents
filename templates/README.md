# Optional prompt templates

Drop a folder here (e.g. `templates/my-domain/`) with `planner.md`, `worker.md`, etc. to append overlays to the base prompts in `prompts/`. Set `TEMPLATE=my-domain` in `.env` or pass `--template my-domain` to `main.py`.

If this directory has no subfolders, OnePrompt uses `prompts/` only.
